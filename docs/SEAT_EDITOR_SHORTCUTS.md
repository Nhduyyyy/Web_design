# Seat Layout Editor - Keyboard Shortcuts

Quick reference guide for all keyboard shortcuts in the Seat Layout Editor.

## 🎨 Tool Selection

| Shortcut | Tool | Description |
|----------|------|-------------|
| `1` | Standard Seat | Place standard theater seats |
| `2` | VIP Seat | Place VIP/premium seats |
| `3` | Couple Seat | Place couple/loveseat seats |
| `4` | Wheelchair Seat | Place wheelchair accessible seats |
| `5` | Aisle | Mark aisle spaces |
| `6` | Stage Area | Mark stage/performance area |
| `D` | Delete Tool | Delete seats by clicking |
| `V` | Select Tool | Select and manipulate seats |

## ✂️ Edit Actions

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+Z` | Undo | Undo last action |
| `Ctrl+Y` or `Ctrl+Shift+Z` | Redo | Redo previously undone action |
| `Ctrl+C` | Copy | Copy selected seats to clipboard |
| `Ctrl+V` | Paste | Paste copied seats (with offset) |
| `Delete` or `Backspace` | Delete Selected | Remove all selected seats |
| `R` | Rotate | Rotate selected seats 90° clockwise |
| `Escape` | Deselect | Clear all selections |
| `Ctrl+A` | Select All | Select all seats in layout |

## 🔍 View Controls

| Shortcut | Action | Description |
|----------|--------|-------------|
| `+` or `=` | Zoom In | Increase canvas zoom level |
| `-` or `_` | Zoom Out | Decrease canvas zoom level |
| `0` | Reset Zoom | Reset zoom to 100% and center view |

## 🖱️ Mouse Actions

| Action | Description |
|--------|-------------|
| **Click** | Place seat (with tool selected) or select seat (with Select tool) |
| **Shift+Click** | Add seat to selection |
| **Ctrl+Click** | Toggle seat in selection |
| **Right-Click** | Delete seat (context menu) |
| **Click+Drag** | Paint multiple seats (with seat tool) |
| **Drag Seat** | Move seat to new position (with Select tool) |

## 💡 Tips

- **Input Detection**: Shortcuts are disabled when typing in input fields or textareas
- **No Conflicts**: Shortcuts are designed to not conflict with browser shortcuts
- **Visual Feedback**: Active tool is highlighted in the toolbar
- **Undo History**: Up to 50 actions can be undone
- **Multi-Select**: Use Shift or Ctrl to select multiple seats before actions

## 🎯 Common Workflows

### Quick Seat Placement
1. Press `1` for standard seats
2. Click or drag on canvas to place seats
3. Press `2` for VIP seats in front rows
4. Continue placing seats

### Editing Layout
1. Press `V` to activate Select tool
2. Click seats to select them
3. Press `R` to rotate
4. Press `Delete` to remove
5. Use `Ctrl+Z` to undo mistakes

### Copy Section
1. Press `V` to select
2. Shift+Click to select multiple seats
3. Press `Ctrl+C` to copy
4. Press `Ctrl+V` to paste (creates offset copy)

### Quick Delete
1. Press `D` for Delete tool
2. Click seats to remove them
3. Or press `V`, select seats, then `Delete` key

## 🚀 Pro Tips

- Use number keys to quickly switch between seat types
- Hold Shift while clicking to build selections
- Use Ctrl+A to select all, then modify properties
- Press Escape frequently to clear selections
- Use Ctrl+Z liberally - undo is your friend!
- Zoom in (`+`) for precise placement
- Zoom out (`-`) to see full layout

## ⚠️ Notes

- Some shortcuts may not work on Mac (use Cmd instead of Ctrl)
- Touch devices use tap and long-press gestures
- Keyboard shortcuts require focus on the editor
- Undo/redo history is cleared when leaving the editor
