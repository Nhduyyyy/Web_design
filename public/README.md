# Static assets (public)

Tất cả file tĩnh dùng trong app. Vite phục vụ từ `/`.

- **masks/** — Ảnh mặt nạ (PNG) dùng cho gallery và fallback.
- **models/** — File 3D (GLB, STL). Chi tiết xem [models/README.md](models/README.md).

Đường dẫn trong code: `/masks/...`, `/models/masks/...`, `/models/objects/...`.
